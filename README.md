# ecode-java
:musical_score: The emoji code utilities for Java

## Requirements
- Java 1.8 or later

### Android
The library also works on Android 4.4 (API level 19) or later.

## Getting started
The library is published on jcenter.

```gradle
repositories {
    jcenter()
}

depepdencies {
    implementation 'ninja.emojigen:ecode:0.1.0'
}
```
